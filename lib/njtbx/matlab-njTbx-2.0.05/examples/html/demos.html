NJ Demos
Try the following demos to know more about the njTBX API and functions usage. If you are viewing the documentation within matlab using matlab help, then you can click the 'Run Matlab Demo' button on individual demo page to execute the code in matlab. Running a demo within matlab is an interactive process, so follow the instructions within matlab in order to complete a demo.

Demos can also be viewed using the 'Demos' feature in matlab. Click on Matlab 'help' and then select 'Demos tab -> Toolboxes -> njTBX'. See a screengrab of the feature.

Note: Some of the demos use functions provided by public libraries 'Seawater' and 'RPSStuff'. Download links to these libraries can be found on njTBX download page.
