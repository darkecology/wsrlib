Method getShape

PURPOSE

mGeoGridVar/getShape - get grid variable shape

SYNOPSIS

function result = getShape(mGeoGridVarObj)

DESCRIPTION 

mGeoGridVar/getShape - get grid variable shape
  
 Method Usage:  
       shape=getShape(mGeoGridVarObj) or
       shape=mGeoGridVarObj.getShape;   
 where,  
       mGeoGridVarObj- mGeoGridVar object. 
 returns,
       shape - matlab array
